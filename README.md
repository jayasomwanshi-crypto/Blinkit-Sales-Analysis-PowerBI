Blinkit Sales Analysis Dashboard
Project Overview

This Power BI project analyzes Blinkit sales data to identify key business insights related to sales performance, customer preferences, outlet performance, and product categories.

The dashboard provides interactive visualizations and KPIs that help stakeholders monitor business performance and make data-driven decisions.

Data Analysis

Total Sales
Average Sales
Number of Items Sold
Average Rating
Business Insights
Sales performance by outlet type
Sales by item category
Outlet size and location analysis
Customer purchasing patterns
Revenue contribution by product category
Outcome

The dashboard helps identify high-performing products and outlets, understand customer preferences, and support strategic business decisions through interactive reporting and visualization.
